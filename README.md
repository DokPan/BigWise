# Установка и Запуск Проекта Try/Catch us

Этот проект использует виртуальное окружение Python и зависимости для работы. Следуйте этим шагам для установки и запуска приложения.

## Шаги:

1. **Откройте папку с проектом**
   - Убедитесь, что у вас есть доступ к папке, где расположен проект.

2. **Добавьте файл с зависимостями**
   - Поместите файл `requirements.txt` в корневую директорию проекта, если его там еще нет. Файл содержит список необходимых библиотек для установки.

3. **Откройте консоль**
   - Запустите терминал или командную строку, в зависимости от вашей операционной системы.

4. **Создайте виртуальное окружение**
   - Введите следующую команду, чтобы создать виртуальное окружение:
     ```bash
     python -m venv .venv
     ```

5. **Активируйте виртуальное окружение**
   - Для Windows используйте команду:
     ```bash
     .venv\Scripts\activate
     ```
   - Для MacOS и Linux используйте команду:
     ```bash
     source .venv/bin/activate
     ```

6. **Установите зависимости**
   - После активации виртуального окружения выполните следующую команду для установки всех необходимых библиотек:
     ```bash
     pip install -r requirements.txt
     ```

7. **Запустите приложение**
   - Введите команду для запуска вашего приложения:
     ```bash
     python mpitapp.py
     ```
```
